# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this directory.

## Directory Overview

React components for the Claudish landing page.

## Components

| Component | Purpose |
|-----------|---------|
| `HeroSection.tsx` | Main hero with headline and CTA |
| `FeatureSection.tsx` | Feature grid showcasing capabilities |
| `BridgeDiagram.tsx` | Visual diagram of Claudish architecture |
| `MultiModelAnimation.tsx` | Animated model switching demo |
| `SmartRouting.tsx` | Smart routing visualization |
| `SupportSection.tsx` | Support and community links |
| `TerminalWindow.tsx` | Terminal UI wrapper component |
| `TypingAnimation.tsx` | Typewriter effect component |
| `BlockLogo.tsx` | Claudish logo component |

## Design Patterns

- Components are self-contained with inline styles or CSS modules
- Animation components use React hooks for state management
- Terminal aesthetics with monospace fonts and dark backgrounds
