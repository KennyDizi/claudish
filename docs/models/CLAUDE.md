# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this directory.

## Directory Overview

Documentation for model selection and configuration.

## Files

| File | Purpose |
|------|---------|
| `choosing-models.md` | Guide to selecting the right model for tasks |
| `model-mapping.md` | Profile-based model mapping configuration |

## Key Concepts

- **Model Selection**: Choose based on task (speed vs reasoning vs vision)
- **Model Mapping**: Map claude-3-* models to OpenRouter alternatives in profiles
- **Context Windows**: Different models have 128K to 2M context limits
