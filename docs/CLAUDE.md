# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this directory.

## Directory Overview

User-facing documentation for Claudish, organized by topic.

## Structure

```
docs/
├── index.md              # Documentation home/overview
├── troubleshooting.md    # Common issues and solutions
├── getting-started/      # Quick start guides
├── usage/                # Usage modes and features
├── models/               # Model selection and mapping
├── advanced/             # Advanced features
└── ai-integration/       # AI agent integration guides
```

## Subdirectories

- `getting-started/` - Installation and quick start
- `usage/` - Interactive mode, single-shot mode, MCP server, monitor mode
- `models/` - Model selection guides, model mapping configuration
- `advanced/` - Automation, cost tracking, environment configuration
- `ai-integration/` - Guides for AI agents using Claudish

## Key Files

| File | Purpose |
|------|---------|
| `index.md` | Main documentation entry point |
| `troubleshooting.md` | Common problems and solutions |

## Writing Documentation

- Use clear, concise language
- Include code examples with proper syntax highlighting
- Link to related documentation
- Keep troubleshooting up to date with common issues
